# Gemini Context Manager (savageai.net)

A High-Fidelity Web Prototype simulating an Android-native Personal Context Management system. 
Built with **ReAct Methodology** (Reasoning + Acting) logic to demonstrate proactive AI assistance.

## Live Demo
:rocket: **[savageai.net](https://savageai.net)**

## Features
- **Context Setting**: Define Role, Goal, and Preferences.
- **ReAct Logic**: Visible "Thinking..." process before responses.
- **Proactive Prompts**: Smart suggestions based on the last interaction.
- **Persistent State**: Simulates app memory using LocalStorage.

## Tech Stack
- HTML5 / CSS3 (Glassmorphism & Mobile-first design)
- Vanilla JavaScript (No heavy frameworks)
- Mock LLM Engine (Simulates Gemini API latency and logic)

## Local Development
1. Clone the repo.
2. Run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open `http://localhost:8000`.
